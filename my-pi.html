The Raspberry Pi

Raspberry Pi 3B+ from Kios Robot https://tokopedia.com/kiosrobot

This will be a headless computer. No need for desktop/etc.

  • Download Raspbian Lite: https://www.raspberrypi.org/downloads/raspberry-pi-os/
  • Install guide: https://www.raspberrypi.org/documentation/installation/installing-images/README.md

Don't need WiFi. Disable wifi, add: dtoverlay=pi3-disable-wifi to file /boot/config.txt - https://blog.sleeplessbeastie.eu/2018/12/31/how-to-disable-onboard-wifi-and-bluetooth-on-raspberry-pi-3/

ISP bandwidth check.

Using Speedtest.net. Details in Does My ISP Sucks https://github.com/ialexs/doesmyispsucks

Scheduled torrent.

Using the transmission-daemon and transmission-cli from Transmission: https://transmissionbt.com/download

  • Pretty straight forward info: https://linuxconfig.org/how-to-set-up-transmission-daemon-on-a-raspberry-pi-and-control-it-via-web-interface
  • Saving download to home NAS: sudo mount -t nfs <NAS_IPADDR>:/nfs/Public /home/pi/torrent/

Findings:

The Transmission configuration file /etc/transmission-daemon/settings.json reseted back to default upon restarting transmission service.

Don't care much. Dirty solution: - Keep a modified settings in another file. Overwrite the configuration file. - Just reload service (don't restart)
