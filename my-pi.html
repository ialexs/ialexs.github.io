Using the transmission-daemon and transmission-cli from Transmission

  • Pretty straight forward info: link
  • Saving download to home NAS: sudo mount -t nfs <NAS_IPADDR>:/nfs/Public /home/pi/torrent/

Findings:

The Transmission configuration file /etc/transmission-daemon/settings.json reseted back to default upon restarting transmission service.

Don't care much. Dirty solution:

  • Keep a modified settings in another file. Overwrite the configuration file.
  • Just reload service (don't restart)
